## 项目的目的

从 patents.google.com 爬取专利数据以供量化研究员研究。

## 项目的步骤思路

1. 下载：按日搜索，下载每天的专利数据 csv 文件。
2. 分析：分析 csv 文件内容，再访问各个专利详情页进一步提取补全信息（如分类，引用关系等）。
3. 入库：将分析后的数据存入数据库。
4. 检查：检查入库数据的引用关系是否完整。
5. 补全：补全缺漏数据。
