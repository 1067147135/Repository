Steps to run my code:
1. Enter folder "119010265_HW_2" in the terminal.
2. Enter "/bin/bash compile_run.sh" and press Enter.
3. See the result.
4. Press key A or D to rotate the triangle.
5. Press key Esc to end the program.

Steps to modify the parameters:
1. Open "Source/parameters.txt".
2. Modify the number without enter any Space. 
3. Run "Build/main" if you have already built it, otherwise run "compile_run.sh".
